Docker 部署速查手册 v1.0
适用：企业知识库 Agent 平台
日期：2026-05

一、基础命令
```bash
# 构建镜像
docker build -t kb-agent:latest .

# 启动服务
docker run -d --name kb-agent \
  -p 8000:8000 \
  -v ./data:/app/data \
  -e DEEPSEEK_API_KEY=sk-xxx \
  kb-agent:latest

# 查看日志
docker logs -f kb-agent

# 停止/重启
docker stop kb-agent && docker start kb-agent

# 进入容器调试
docker exec -it kb-agent bash
```

二、Docker Compose 编排
```yaml
version: '3.8'
services:
  kb-agent:
    build: .
    ports:
      - "8000:8000"
    volumes:
      - ./data:/app/data
    environment:
      - DEEPSEEK_API_KEY=${DEEPSEEK_API_KEY}
      - LLM_PROVIDER=deepseek
    restart: unless-stopped
```

三、常用排错
1. 端口冲突：lsof -i :8000 查看占用进程
2. 磁盘满：docker system prune -a 清理无用镜像
3. 容器无法启动：docker logs kb-agent --tail 50
4. ChromaDB 数据损坏：删除 data/chroma_db 后重启容器

四、生产环境建议
- 使用 docker-compose 管理多服务
- 数据目录挂载到宿主机（防止容器删除后数据丢失）
- 设置资源限制：--memory="2g" --cpus="2"
- 使用 Docker Hub 或私有 Registry 管理镜像版本
