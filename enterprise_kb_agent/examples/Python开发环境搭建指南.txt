Python 开发环境搭建指南 v2.1
适用：macOS / Linux 新员工入职
日期：2026-04

一、安装 Python
```bash
# macOS（推荐用 Homebrew）
brew install python@3.12

# 验证
python3 --version  # 应输出 3.12.x
```

二、虚拟环境管理
```bash
# 创建项目虚拟环境
python3 -m venv .venv

# 激活（macOS/Linux）
source .venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 退出
deactivate
```

三、常用工具链
- 包管理：pip / uv（更快）
- 格式化：black --line-length=100 .
- 导入排序：isort .
- 类型检查：mypy app/
- 测试：pytest -q --cov=app/

四、VS Code 配置
```json
{
  "python.defaultInterpreterPath": ".venv/bin/python",
  "python.linting.enabled": true,
  "editor.formatOnSave": true,
  "python.formatting.provider": "black"
}
```

五、常见问题
1. pip install 报 SSL 错误 → 检查公司代理设置
2. virtual env 激活后仍用系统 python → 检查 PATH 顺序
3. ChromaDB 需要 sqlite3 >= 3.35 → macOS 自带版本过低需升级
4. pypdf 解析中文 PDF 乱码 → 确认 PDF 字体嵌入
